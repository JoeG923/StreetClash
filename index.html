Mini Street Clash

Fight Controls

P1: A/D move, W jump, C punch, V kick

P2: Left/Right move, Up jump, / punch, Down kick

Gamepads: D-pad/Left Stick move, Y/△ jump, A/X punch, B/O kick, Start begin/restart.

Edit window.GAME_CONFIG in this file to remap keys or swap controller slots.

Press R to start or restart a round.

Press F for fullscreen.

Jump then kick to trigger a jump kick. First to zero life loses.
